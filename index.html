Personal Portfolio

This project marks my first step into web development, where I created a single-page website using only HTML and CSS. The goal was to build a responsive webpage featuring a navigation bar, an "About Me" section, a project showcase, and a dedicated photography gallery to highlight my passion for photography. I included a personal profile picture, project-related images, and a collection of my favorite nature and travel photos to make the portfolio visually engaging and reflective of my interests. This project helped me solidify my understanding of basic web development concepts, responsive design principles, and the importance of combining technical skills with personal creativity.
